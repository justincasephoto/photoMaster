cómo configurar el proyecto + ver
  1. Google Drive: sube las fotos, acceso público. URL directa: drive.google.com/uc?export=view&id=TU_ID
  2. Tally.so: crea un form con campos name, story, photographer. Activa email notifications.
  3. Por cada persona: duplica index.html, edita el objeto SESSION, guarda en /sessions/nombre-lugar/index.html.
  4. Sube a GitHub: git add . && git commit -m "sesión" && git push — Vercel despliega en 30 segundos.
  5. Genera el QR aquí y muéstralo en tu teléfono en la calle.
