cómo configurar el proyecto + ver
  1. Edita index.html — en la sección SESSION, cambia photographer, phone_blik y tally_form_url.
  2. Google Drive: sube las fotos, dales acceso público, y copia el ID del archivo. La URL directa tiene el formato: drive.google.com/uc?export=view&id=TU_ID
  3. Tally.so: crea un form con campos name, story, photographer. Activa notificaciones por email. Gratis.
  4. Vercel: conecta tu repositorio de GitHub y despliega. Cada carpeta /sessions/marta-lazienki/ con su index.html es una URL automática.
  5. Para cada persona: duplica index.html, edita el objeto SESSION, guárdalo en /sessions/nombre-lugar/index.html.
  6. Usa este panel admin para generar el QR y el link. Muéstralo en tu pantalla o imprímelo en tarjetas.
